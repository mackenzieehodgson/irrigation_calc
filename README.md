# Irrigation Calculator

Farm irrigation energy calculator built with Voila and Python.

## 🚀 Try the App

### Option 1: Run on Binder (Free, No Installation)
Click the link below to run the app in your browser:

[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/mackenzieehodgson/irrigation_calc/HEAD?urlpath=voila%2Frender%2Firrigation_calc_voila_app.ipynb)

### Option 2: Run Locally
1. Clone this repository:
   ```bash
   git clone https://github.com/mackenzieehodgson/irrigation_calc.git
   cd irrigation_calc
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the Voila app:
   ```bash
   voila irrigation_calc_voila_app.ipynb
   ```

The app will open in your browser at `http://localhost:8866`

## 📋 Files

- `irrigation_calc_voila_app.ipynb` - Main Voila interactive app
- `requirements.txt` - Python dependencies

## 📝 About

This tool calculates irrigation energy requirements for farm operations.
